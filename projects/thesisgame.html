A programming game made as part of my Master thesis during the spring of 2023. In the game the player controls a robot through a visual programming language. During this project I primarily worked on the programming aspect of the game. I made the backend logic, where commands in the user interface is translated to code that the robot can execute. I also made the visual front-end of the programming interface.
